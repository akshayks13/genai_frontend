### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/akshayks13/genai_frontend
   cd genai_frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx next dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)


### Available Scripts

```bash
# Development
```
